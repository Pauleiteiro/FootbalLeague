# Terças FC tabela semanal

Vai servir para automatizar o sistema atual da tabela semanal dos jogos realizados às terças

## Regras e sistemas de pontos

-Frequência: Os jogos ocorrem todas as terças-feiras
-Escolha de equipas: As escolhas são feitas pelo 1º e 2º classificados
-Pontos: 
    - Vitória: +3 pontos
    - Empate: +2 pontos
    - Derrota: +1 ponto
    - Último jogo do campeonato: Dobro dos pontos
    - Falta de comparência: -3 pontos
-Critério de desempate: Maior número de jogos realizados
-Eligibilidade: Os jogadores têm de participar em pelo menos 50% dos jogos


## Tecnologias utilizadas

-Linguagem: Python
-Base de dados: SQLIte (Dev) / PostgreSQL (Prod)
-Framework: FastAPI
